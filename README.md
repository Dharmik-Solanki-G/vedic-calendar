# Vedic Calendar 2026

A beautiful, interactive web-based Vedic calendar displaying important Hindu dates and events for 2026.

## Features

- **Monthly Calendar View** - Clean, minimalist design showing all dates
- **Bhadra Periods** - Displays when Bhadra begins and ends throughout the year
- **Purnima (Full Moon)** - Shows all full moon dates
- **Amavasya (New Moon)** - Shows all new moon dates  
- **Dwipushkar Yoga** - Displays Dwipushkar Yoga days
- **Indra Yoga** - Shows Indra Yoga dates
- **Interactive Notes** - Click any date to see detailed information
- **PDF Download** - Download any month as PDF
- **Print Support** - Print-friendly layout

## Live Demo

Visit the calendar at: [GitHub Pages](https://dharmik-solanki-g.github.io/vedic-calendar/)

## Data Sources

All dates are hardcoded from:
- `Bhadra dates 2026.txt` - Bhadra periods for the year
- `2026 Purnima Dates.txt` - Full moon dates
- `2026 Amavasya Dates.txt` - New moon dates
- `Dwipushkar Yoga Days.txt` - Dwipushkar Yoga timings
- `Indra Yoga Dates.txt` - Indra Yoga dates

## Usage

1. Open `index.html` in your browser
2. Navigate between months using the arrow buttons
3. Click any date to see detailed information in the notes section
4. Click "Download" to save the current month as PDF
5. Use browser print function to print the calendar

## Technologies

- Pure HTML, CSS, and JavaScript
- jsPDF for PDF generation
- html2canvas for PDF rendering
